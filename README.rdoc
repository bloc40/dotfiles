=== To Install

  $ git clone git://github.com/bloc40/dotfiles.git
  $ cd dotfiles
  $ ./install
  $ git clone https://github.com/gmarik/Vundle.vim.git ~/.vim/bundle/Vundle.vim
  $ vim +PluginInstall +qall

=== To remove the symlinks

  $ cd dotfiles
  $ ./remove_symlinks

=== To Update

  $ cd dotfiles
  $ git pull --rebase
