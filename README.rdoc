= Jamal El Milahi's Dot Files

== Pre-requisites

You need to have Macvim[http://code.google.com/p/macvim/] and Janus[https://github.com/carlhuda/janus] installed.

== To Install

  $ git clone git://github.com/bloc40/dotfiles.git
  $ cd dotfiles
  $ ./install

== To remove the symlinks

  $ cd dotfiles
  $ ./remove_symlinks

== To Update

  $ cd dotfiles
  $ git pull --rebase
