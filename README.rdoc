= Jamal El Milahi's Dot Files

== Pre-requisites

You need to have Macvim[http://code.google.com/p/macvim/], Janus[https://github.com/carlhuda/janus], and Oh-My-Zsh[https://github.com/robbyrussell/oh-my-zsh] installed.

== To Install

  $ git clone git://github.com/bloc40/dotfiles.git
  $ cd dotfiles
  $ ./install

== To Update

  $ cd dotfiles
  $ git pull --rebase
